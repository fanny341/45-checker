GP45 DB v1.1.0 - Simplified UI
🔍 CARI BARANG
Ketik di kotak search. Cari pake nama atau barcode — otomatis detek. Multi-kata (AND): wipro kunci sok 8. Tap item untuk lihat detail harga, stok, & stok cabang.
📷 SCAN BARCODE
Tap ikon 📷 di pojok kiri header. Arahkan ke barcode. Hasil langsung tampil.
📋 OFFLINE · 🌐 ONLINE
OFFLINE = data lokal HP. Cepet, gak pakai internet. ONLINE = langsung ke server 200 (butuh WiFi toko).
📋 SALIN NAMA + HARGA
Tap tombol 📋 di detail barang. Otomatis copy Nama Barang - RpXX.XXX ke clipboard. Tinggal paste kirim ke customer.
🔄 SYNC DATA
Menu ☰ → Sync Data. Update database offline dengan data terbaru dari server 200. Butuh koneksi WiFi ke jaringan toko. Bisa dibatalkan kalo salah pencet.
💰 PRICE CARD
Menu ☰ → Price Card. Buat daftar barang untuk print label. Cari barang, set qty, simpan. Lihat & hapus di riwayat.
📈 OMSET
Menu ☰ → Omset. Lihat total penjualan per sales per hari. Tap nama sales untuk lihat barang yg dijual.
🌙 DARK MODE
Menu ☰ → Dark Mode. Toggle tema gelap/terang. Tersimpan otomatis.
